# 长安街跑者

一个原创低多边形 3D 无尽跑酷网页游戏，使用 HTML、CSS、JavaScript 和 Three.js 制作。

## 本地运行

```powershell
python -m http.server 5177 --bind 127.0.0.1
```

然后打开：

```text
http://127.0.0.1:5177/
```

## GitHub Pages

把本目录作为仓库根目录发布。进入 GitHub 仓库的 `Settings -> Pages`，选择 `main` 分支和根目录发布即可。
